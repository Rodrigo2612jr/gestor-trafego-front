Política de Privacidade

Última atualização: março de 2026

O Gestor Leo IA ("aplicativo") é uma ferramenta de gestão de tráfego pago que se conecta à API de Marketing da Meta para criar e gerenciar campanhas publicitárias.

Dados coletados

O aplicativo acessa dados da sua conta de anúncios do Meta Ads (campanhas, conjuntos, anúncios e métricas de desempenho) exclusivamente para exibição e gerenciamento interno. Nenhum dado é compartilhado com terceiros.

Uso dos dados

Os dados são usados apenas para: criação e gerenciamento de campanhas, exibição de métricas e geração de relatórios internos para o próprio titular da conta.

Armazenamento

Tokens de acesso são armazenados de forma segura e usados exclusivamente para autenticação com a API da Meta.
